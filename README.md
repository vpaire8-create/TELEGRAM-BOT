# 🤖 SYAPA KING - Facebook Automation Bot

## 📌 Features
- ✅ Facebook Group & Inbox Automation
- ✅ Cookies-based Authentication (No Token Expiry)
- ✅ Continuous Message Loop
- ✅ Multi-user Support
- ✅ Admin Approval System
- ✅ Encrypted Cookie Storage
- ✅ Real-time Status Updates

## 📦 Requirements

### System Requirements
- Python 3.8+
- Chromium/Chrome browser
- 512MB RAM minimum
- Linux/Ubuntu (Recommended)
